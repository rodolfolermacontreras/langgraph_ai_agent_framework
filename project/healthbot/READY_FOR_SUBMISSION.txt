## ğŸŒŸ Stand-Out Feature: Dynamic Question Generation

### What It Does
Users can request **multiple quiz questions on the same health topic** without restarting the conversation. Each question is **freshly generated** by the LLM, not pre-baked or memorized.

### Implementation
```
[User asks new topic]
    â†“
[Summary is generated]
    â†“
[Quiz Question 1]
    â†“
[User selects: "More questions"]
    â†“
[Quiz Question 2] â† FRESH question, same topic, same summary
    â†“
[User selects: "More questions"]
    â†“
[Quiz Question 3] â† DIFFERENT perspective, same topic
    â†“
[User selects: "New topic" or "Exit"]
    â†“
[State resets, returns to topic selection]
```

### Why It's Valuable
- **Prevents memorization** by generating unique questions
- **Deeper learning** through multiple perspectives
- **Better assessment** by testing different aspects of knowledge
- **Maintains engagement** without conversation reset
- **State management** keeps topic context while switching content
